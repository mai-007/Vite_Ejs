# vite frontEndDevEnvironment
Viteでプレーンなhtml制作パッケージ
Ejs

## what this can do
- HTMLをEjsで書く
- Sassコンパイル
- Jsバンドル

# Quick Start
* npmのインストール
   * `npm install`
* 開発サーバーの立ち上げ
  * `npm run dev`
* ファイルのビルド
  * `npm run build`

# 画像の取り扱いについて
変換かけたくないものは[public]フォルダへ入れる事
